// Befehl zum Kopieren des öffentlichen SSH-Schlüssels auf einen entfernten Server

{

    ~ssh-kopieren-schlüssel
    ~scp
    ~ssh-send-key
}

// Zweck des Kopierens des öffentlichen SSH-Schlüssels auf einen entfernten Server mit ssh-copy-id
Was ist der Zweck des Kopierens Ihres öffentlichen SSH-Schlüssels auf einen entfernten Server mit ssh-copy-id?
{
    =Um sich beim entfernten Server ohne Passwort zu authentifizieren.
    ~Um Dateien zwischen dem lokalen und dem entfernten Rechner zu übertragen.
    ~Um Ihren privaten Schlüssel mit anderen zu teilen.
    ~Um die SSH-Konfiguration des Servers zu ändern.
}

// Zu bearbeitende Datei für die Konfiguration der SSH-Optionen auf dem lokalen Rechner

{

    ~/etc/ssh/sshd_config
    ~/etc/ssh/ssh_config
    ~~/.ssh/authorized_keys
}

// Zweck der Datei known_hosts in SSH
Was ist der Zweck der Datei known_hosts in SSH?
{
    =Zur Verfolgung der Host-Schlüssel des entfernten Servers.
    ~Um private Schlüssel zu speichern.
    ~Auflistung autorisierter Benutzer.
    ~Um öffentliche Schlüssel zu speichern.
}
