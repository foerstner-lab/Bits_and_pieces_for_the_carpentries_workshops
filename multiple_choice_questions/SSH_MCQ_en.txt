Which command is used to generate SSH key pairs on a local machine?
A) ssh-genkey
B) ssh-generate
C) ssh-keygen
D) ssh-encrypt

Correct Answer: C) ssh-keygen

What is the default name for the private SSH key file generated by ssh-keygen?
A) id_rsa.pub
B) id_rsa
C) private_key.pub
D) public_key

Correct Answer: B) id_rsa

Which command is used to copy your SSH public key to a remote server?
A) ssh-copy-key
B) scp
C) ssh-copy-id
D) ssh-send-key

Correct Answer: C) ssh-copy-id

What is the purpose of copying your SSH public key to a remote server using ssh-copy-id?
A) To transfer files between the local and remote machines.
B) To share your private key with others.
C) To authenticate to the remote server without a password.
D) To change the server's SSH configuration.

Correct Answer: C) To authenticate to the remote server without a password.

Which file should you edit to configure SSH options on your local machine?

A) /etc/ssh/sshd_config
B) ~/.ssh/config
C) /etc/ssh/ssh_config
D) ~/.ssh/authorized_keys

Correct Answer: B) ~/.ssh/config

What is the purpose of the known_hosts file in SSH?
A) To store private keys.
B) To list authorized users.
C) To track remote server host keys.
D) To store public keys.

Correct Answer: C) To track remote server host keys.

Which SSH command is used to specify a different identity file (private key) when connecting to a remote server?
A) ssh -i
B) ssh -p
C) ssh -k
D) ssh -r

Correct Answer: A) ssh -i

Which SSH command allows you to securely copy files between your local machine and a remote server?
A) ssh-copy-id
B) scp
C) ssh-sync
D) ssh-move

Correct Answer: B) scp
