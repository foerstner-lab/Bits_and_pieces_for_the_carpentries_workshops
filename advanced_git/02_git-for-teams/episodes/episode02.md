# Episode 2 - Preparation

- Objective: Prepare everything for live coding
- Estimated time: 30min
- Teaching techniques: Live Coding

## Prepare Git

- Open a new shell which allows you to use the Git client.
- Create the working directory: `mkdir git-and-gitlab-for-teams`

### Configure your Git Identity

- `git config --global user.name "YOUR NAME"`
- `git config --global user.email "YOUR EMAIL"`

> You can use [a private email address](https://docs.gitlab.com/ce/user/profile/#private-commit-email) generated by GitLab as well.

### Create a Git Shortcut

- `git config --global alias.graph "log --all --decorate --oneline --graph"`
- Explain that we just created the command alias `git graph`.

## Prepare Workshop Materials

- `git clone https://gitlab.com/hifis/hifis-workshops/software-development-for-teams/workshop-materials.git`
- This repository contains materials the we use for later exercises.

## Fork the Example GitLab Project

- Show project page of the example repository: https://gitlab.com/hifis/hifis-workshops/software-development-for-teams/workshop/sample-calculator-iteration-1
- Click `Fork` on the right side of the GitLab project page (Next to `Clone`).
- Fork the GitLab project into your private namespace.
- Clone from new location: `git clone https://gitlab.com/YOURUSERNAME/sample-calculator-iteration-1.git`

## Key Points

- We prepared everything to start with the exercises.
